- FluidSynth:
	- for Windows:
		- lib name: fluidsynth.dll
		- project website: http://www.fluidsynth.org/
		- fluidsynth.dll should be placed in PATH or in this lib directory (will be added to PATH when running start scripts)
		- you can compile it yourself from http://sourceforge.net/projects/fluidsynth/files/fluidsynth-1.1.3/ 
		or download one from here http://svn.drdteam.org/zdoom/fluidsynth.7z (ofcourse download at your own risk, link found via stackoverflow).
